# Simple Contact Book (Python)

A beginner-friendly **menu-driven console app** that stores contacts using a Python **dictionary** (`name -> phone number`).
It also uses a **set** to track unique phone numbers.

## Requirements
- Python 3.x

## How to Run
In your terminal (inside the project folder):

```bash
python Mini_Project-2_Simple_Contact_Book_Updated.py
```

## Menu Options
1. View all contacts  
2. Add new contact  
3. Search contact  
4. Delete contact  
5. Count total contacts  
6. Show all names only  
7. Show all phone numbers only  
8. Exit  

## Data Structures Used
- **Dictionary**: `contacts = {"Name": "Phone"}` for storing contacts.
- **Set**: `unique_numbers` for keeping phone numbers unique.
